# Extension Assets

This directory contains assets for the TerminalWON Kiro extension marketplace listing.

## Required Files

### icon.png
- **Size**: 128x128 pixels (minimum), 256x256 recommended
- **Format**: PNG with transparency
- **Design**: 
  - Single terminal window icon with streaming indicator
  - Gradient from indigo (#6366f1) to emerald (#10b981)
  - Kiro-inspired styling
  - Works on light and dark backgrounds

### gallery/ (Optional)
Screenshots for the marketplace gallery:

1. **live-streaming.png** (1280x800)
   - Show: Live terminal streaming in action

2. **kiro-integration.png** (1280x800)
   - Show: Deep Kiro IDE integration

3. **remote-terminals.png** (1280x800)
   - Show: Remote terminal creation and management

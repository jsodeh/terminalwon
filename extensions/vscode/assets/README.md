# Extension Assets

This directory contains assets for the TerminalWON VSCode extension marketplace listing.

## Required Files

### icon.png
- **Size**: 128x128 pixels (minimum), 256x256 recommended
- **Format**: PNG with transparency
- **Design**: 
  - Single terminal window icon
  - Gradient from indigo (#6366f1) to emerald (#10b981)
  - Minimalist, recognizable at small sizes
  - Works on light and dark backgrounds

# Extension Assets

This directory contains assets for the TerminalWON Cursor extension marketplace listing.

## Required Files

### icon.png
- **Size**: 128x128 pixels (minimum), 256x256 recommended
- **Format**: PNG with transparency
- **Design**: 
  - Single terminal window icon with AI indicator
  - Gradient from indigo (#6366f1) to emerald (#10b981)
  - Cursor-inspired styling
  - Works on light and dark backgrounds

### gallery/ (Optional)
Screenshots for the marketplace gallery:

1. **ai-chat-continuity.png** (1280x800)
   - Show: AI conversation continued across devices

2. **cursor-integration.png** (1280x800)
   - Show: Cursor-specific AI features and insights

3. **mobile-access.png** (1280x800)
   - Show: Phone with AI chat continuation
